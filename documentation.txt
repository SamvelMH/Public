Проект предостовляет сервис по предоставлению аналитики по открытым данным
из источников ВЦИОМ.
Сервис позволяет выбирать интересующие категории и выполнять запрос данных по ним.
Сервис демонстрирует аналитику в виде показателей:
Максимальное/минимальнео значение
Быстрорастущие категории.
Топ по количеству.
Топ по ключевому показателю.
График в разрезе выбранного период, с возможностью выбора интересующего периода для
быстрого просмотра данных.
Таблица аналити по выбранному периоду данных.
Распределение выбарнных категорий по географическому показателю.
Выгрузка аналитики в excell и pdf.
